## 🇵🇹 Português

### O que é este dashboard?

Este é o **PMO Dashboard oficial da BridgePoint Africa Europa** — uma ferramenta interna de gestão de projetos desenvolvida para coordenar, acompanhar e documentar o progresso do projeto **Jardins de Leiria Residence** em Icolo e Bengo, Angola.

O dashboard foi criado para uso exclusivo da equipa BridgePoint e do promotor do projeto, Carlos Simões / JTA. Funciona como o centro de controlo da relação entre a BridgePoint e o promotor, garantindo transparência total em todas as fases do projeto.

### Para que serve?

A BridgePoint Africa Europa atua como **parceiro estratégico PMO** neste projeto — não como vendedor de imóveis, mas como conector entre capital europeu e oportunidades em Angola, Portugal e África.

Este dashboard permite:

- Acompanhar em tempo real o estado de cada tarefa e documento do projeto
- Identificar claramente o que está concluído, em curso ou em aberto
- Mostrar ao promotor exatamente o que a BridgePoint aguarda para poder avançar
- Documentar todas as decisões, reuniões e riscos num único lugar
- Preparar o projeto para due diligence internacional e apresentação a investidores

### O que contém?

| Módulo | Descrição |
|--------|-----------|
| **Visão Geral** | Painel executivo com KPIs e barra de prontidão para investidores |
| **Funções** | Distribuição clara de responsabilidades entre BridgePoint e promotor |
| **Entregas do Promotor** | Lista de documentos que a BridgePoint aguarda de Carlos Simões |
| **Estado do Projeto** | Estado detalhado por categoria — jurídico, técnico, financeiro, PMO |
| **Matriz de Ações** | Rastreamento de tarefas por responsável, prazo e estado |
| **Registo de Documentos** | Controlo de versões e estado de todos os documentos do projeto |
| **Registo de Riscos** | Identificação, avaliação e mitigação de riscos PMO |
| **PMO Tracker** | Prioridades 1 a 4 com checkboxes e barras de progresso |
| **Roadmap** | Jornada completa do projeto até ao financiamento |
| **Registo de Decisões** | Log de todas as decisões tomadas com data e responsável |
| **Registo de Reuniões** | Notas e atualizações de cada reunião com o promotor |

### Tecnologia

- Aplicação web de página única (**Single Page Application**)
- HTML, CSS e JavaScript puro — sem frameworks externos
- Dados guardados no browser (**localStorage**) — sem servidor necessário
- Totalmente bilingue: **Português Europeu** (idioma principal) e **Inglês**
- Compatível com qualquer browser moderno
- Funciona offline após carregamento inicial

## 🇬🇧 English

### What is this dashboard?

This is the **official BridgePoint Africa Europa PMO Dashboard** — an internal project management tool built to coordinate, track and document the progress of the **Jardins de Leiria Residence** project in Icolo e Bengo, Angola.

The dashboard is designed for exclusive use by the BridgePoint team and the project developer, Carlos Simões / JTA. It serves as the control centre for the relationship between BridgePoint and the developer, ensuring full transparency across all project phases.

### What is it for?

BridgePoint Africa Europa acts as a **strategic PMO partner** on this project — not as a property seller, but as a connector between European capital and opportunities in Angola, Portugal and Africa.

This dashboard enables:

- Real-time tracking of the status of every task and document in the project
- Clear visibility of what is completed, in progress or open
- Showing the developer exactly what BridgePoint is waiting for in order to proceed
- Documenting all decisions, meetings and risks in one place
- Preparing the project for international due diligence and investor presentation

### What does it contain?

| Module | Description |
|--------|-------------|
| **Overview** | Executive dashboard with KPIs and investor readiness progress bar |
| **Roles** | Clear responsibility split between BridgePoint and the developer |
| **Developer Deliverables** | List of documents BridgePoint is waiting for from Carlos Simões |
| **Project Status** | Detailed status by category — legal, technical, financial, PMO |
| **Action Matrix** | Task tracking by owner, due date and status |
| **Document Register** | Version control and status of all project documents |
| **Risk Register** | PMO risk identification, assessment and mitigation |
| **PMO Tracker** | Priorities 1 to 4 with checkboxes and progress bars |
| **Roadmap** | Full project journey from structuring to funding |
| **Decision Log** | Log of all decisions made with date and owner |
| **Meeting Log** | Notes and updates from every meeting with the developer |

### Technology

- Single Page Application (SPA)
- Pure HTML, CSS and JavaScript — no external frameworks
- Data stored in the browser (**localStorage**) — no server required
- Fully bilingual: **European Portuguese** (primary language) and **English**
- Compatible with any modern browser
- Works offline after initial load
